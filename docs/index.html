1 Open Analytics Resources for Health and Care

Originally a Google document is supported by the Association of Professional Healthcare Analysts (ApHA), This Equals and Keyah Consulting, in collaboration with the Health Foundation, released under a CC-BY-SA licence. Attributions to ApHA 2020. This document is still active.

This document is a resource for analysts and data scientists working in the health and care sectors primarily, but not exclusively, in the UK.

1.1 Technical notes

This {bookdown} is available for review, curation, comment and update from anyone who wishes to do a pull request or an issue. Using GitHub means this is version controlled, is backed up and is available to everyone.

Translation from the Google document was via a copy to a personal account and the use of the Google Add-On for turning docs to markdown.
